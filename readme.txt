## Pre-Requisite for running the Web Project :
 - Node should be installed (Download Link: https://nodejs.org/en/)
 - Any Browser

## Please follow the below steps for installation :
 - Download the project form GIT : https://github.com/vaibhsir1991/MOJO
 - Unzip the file in a new folder
 - Go to Command Prompt 
 - Go to the project folder
 - Run `npm install` 
 - Run `npm start`
 - App will start running on `http://localhost:5000/#/`
